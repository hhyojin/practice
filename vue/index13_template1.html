뷰 디렉티브: html 태그 안에 v- 접두사를 가진 것들

v-if. 지정한 데이터의 참거짓에 따라 해당 html 요소를 표시하거나 표시하지 않음
  • v-for. 지정한 뷰 데이터 개수만큼 해당 html 태그를 반복 출력
  • {{ system }}

v-show. v-if와 비슷하게 진위여부에 따라 html 태그를 표시하거나 표시하지 않음. 다만 v-if는 태그를 완전히 삭제하지만 v-show는 실제 태그는 남아 있고 display:none임

v-bind. html 태그의 기본 속성과 뷰 데이터 속성을 연결

v-on. 화면 요소의 이벤트를 감지하여 처리.

v-model. form에서 주로 사용되는 속성. 폼에 입력한 값을 뷰 인스턴스 데이터와 즉시 동기화. 화면에 입력한 값을 저장해 서버에 보내거나 watch와 같은 고급속성을 이용해 추가 로직 수행 가능. input,select, textarea 태그에만 사용 가능.
